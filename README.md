# 文言文bot
使用[node-telegram-bot-api](https://github.com/yagop/node-telegram-bot-api)开发的Telegram bot。  
目前使用百度翻译API实现基本白话文言互译功能，未来将调用其他API实现更多古诗文相关功能。
